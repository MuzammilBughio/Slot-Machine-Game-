Python Slot Machine Game
Project Description

A console-based slot machine game built in Python where users can deposit money, place bets on multiple lines, and spin to win. The game features dynamic payouts based on symbol matches, tracks user balance, and provides interactive gameplay.

Features

Deposit and manage your virtual balance

Bet on 1–3 lines per spin

Dynamic payout system based on matching symbols

Real-time display of slot machine results

Shows winning lines and total winnings per spin

Interactive command-line interface

Technologies Used

Programming Language: Python

Libraries: random (built-in)

How to Play

Clone or download the repository.

Run the Python script:

python slot_machine.py


Enter the amount to deposit.

Choose the number of lines to bet on (1–3).

Place your bet per line.

Press Enter to spin the slot machine.

View your winnings and updated balance.

Repeat or quit by entering q.

Game Mechanics

The slot machine has 3 rows and 3 columns.

Symbols have different counts and values:

A: 2 symbols, 5x payout

B: 3 symbols, 4x payout

C: 6 symbols, 3x payout

D: 8 symbols, 2x payout

Wins occur when all symbols in a line match.

Total bet = bet per line × number of lines.

Net winnings = total winnings − total bet.
